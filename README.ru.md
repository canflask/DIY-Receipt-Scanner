# САМОДЕЛЬНЫЙ сканер квитанций

[![Лицензия: AGPL-3.0](https://img.shields.io/badge/License-AGPL--3.0-blue.svg)](https://www.gnu.org/licenses/agpl-3.0.en.html)

**САМОДЕЛЬНЫЙ сканер квитанций** - это сканер квитанций в полоску с автоматической регистрацией и локальным веб-сервером.  
Проект полностью основан на открытом исходном коде и сочетает в себе 3D-печать, электронику, компьютерное зрение и веб-разработку.

---

## 🎯 Цель проекта

Создать точный и доступный по цене сканер чеков, который можно собрать дома или в мастерской.  
Основные функции:

- Автоматическая подача квитанции с помощью роликового механизма
- Полосовое сканирование для высокоточного распознавания текста
- Локальный веб-интерфейс для просмотра текста и базовой аналитики
- возможность интеграции с мобильными приложениями и будущим ML-анализом квитанций

## ❗️ Необходимые навыки участников

- **Механика / электроника:** Автоматическая подача, управление шаговым двигателем, датчики  
- **Python / компьютерное зрение: ** обработка изображений в виде полосок, распознавание текста, склеивание полосок  
- **Интерфейс / Веб:** локальный интерфейс, аналитика квитанций  
- **ML (опционально):** проверка классификации, улучшенное распознавание

## 🛠 Минимальный набор компонентов

- Raspberry Pi 5 + microSD 32-64 ГБ  
- Модуль камеры Raspberry Pi 3  
- Шаговый двигатель NEMA 17 + ДРАЙВЕР (A4988 или TMC2208)  
- Резиновые ролики, валы, подшипники  
- Инфракрасные датчики прерывания луча (наличие/выход сигнала)  
- Светодиодная лента 12 В (4000-5000 К) + источник питания  
- Печатные детали (PLA/PETG): корпус, туннель, держатели роликов, крепление для камеры, лотки

*(Полный список компонентов и обновлений смотрите в разделе [BOM.md ](./BOM.md ))*

## 🖥 Установка и запуск программного обеспечения

1. Установите **Raspberry Pi OS** на карту microSD  
2. Подключите драйверы камеры и двигателя  
3. Установите зависимости:

```
sudo apt update
sudo apt install python3-pip tesseract-ocr
pip install fastapi uvicorn opencv-python pytesseract sqlalchemy
```
Запустите сервер:
```
uvicorn api:app --host 0.0.0.0 --port 8000
```
Откройте веб-интерфейс: http://<IP_RaspberryPi>:8000
